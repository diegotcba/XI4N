gauges
======

An outgauge plugin that draws a beatiful gauge on a webpage, representing whats
happening on the car you're driving (in real time).

Designed to be used in conjunction with a tablet or smart phone.

================ ====
Capability       Y/N
================ ====
InSim            Y
OutSim           N
OutGauge         Y
LFSW Relay       N
Options          Y
Multi-connection N
Dependencies     N  
================ ====

-------
