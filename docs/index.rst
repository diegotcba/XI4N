xi4n: eXtensible Insim for Nodejs
=================================

Welcome to the documentation for xi4n, both a platform and library that can be
used with Nodejs to talk to Insim, Outsim, Outgauge and the LFS World Relay.
If you're not sure what these are then this may not be the application for you.

If you're new to xi4n, begin with the :doc:`/guides/main` guide, which walks you
through installing xi4n, and configuring it.
